### 目前完成功能

+ 歌曲：播放、拖动进度、歌词滚动、音量调整、下载、播放列表
+ 发现页：推荐、歌单、歌手、排行榜
+ 最近播放
+ 喜欢的音乐
+ 登录：登录，退出登录
+ 歌单详情页：歌曲列表，歌单页搜索
+ 专辑详情页：歌曲列表，搜索
+ 歌手详情页：专辑列表，歌手描述
+ 搜索：歌曲搜索
+ 视频：视频列表
+ 部分页面移动端适配
